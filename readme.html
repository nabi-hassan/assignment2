This is Readme.

This website is uploaded on github and github pages is used for hosting this site

1) To host on github pages you need to create a new repository on github
2) Set repository to public
3) Upload your Project with index.html file on main page
4) Goto Settings and scroll down to Github pages section
5) Click on sources button and set to main and then click save
6) Wollah! your static website is hosted
